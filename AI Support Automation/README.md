# AI-Support-Automation

This project demonstrates a Python automation workflow for handling customer support queries.

## Features
- Automatically generates responses to tickets based on query content
- Categorises tickets by type
- Outputs responses to a CSV file

## Skills Demonstrated
- Python scripting
- Automation of repetitive tasks
- Problem-solving and experimentation
